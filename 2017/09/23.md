* `for`(范围)
    * 原理
        ```C++
        for (auto &r : iv) {
            r *= 2;
        }
        for (auto beg = iv.begin(), end = iv.end(); beg != end; ++beg) {
            auto &r = *beg;
            r *= 2;
        }
        ```
