## VIM

### Environment
* Ubuntu
* VIM 7.4

### Modes
* Insert Mode
* Command Mode
    * Visual Mode

### Shortcuts
* Command Mode
    * `:q`
        * close
    * `:w`
        * save
    * `:wq` or `x`
        * save and close
    * `:q!`
        * close and don't save
    * `v`
        * Enter visual mode for selection of **Characters**
    * `V`
        * Enter visual mode for selection of **Lines**
    * `ctrl-v`
        * Enter visual mode for selection of **Blocks**
    * `y`
        * yank/copy selected region
    * `yy`
        * yank/copy entire line
    * `c`
        * cut selection
    * `p`
        * paste yank content 
    * `u`
        * undo
    * `ctrl-r`
        * redo
