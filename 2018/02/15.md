## <<Effective C++>>

#### Item 1
* View C++ as a federation of languages
* sublanguage
    * C
        * blocks
        * statements
        * preprocessor
        * built-in data types
        * arrays
        * pointers
        * ...
    * Object-Oriented C++
        * **C with classes**
            * classes(constructor and destructor)
            * encapsulation
            * inheritance
            * polymorphism
            * virtual function
            * ...
    * Template C++
        * **Generic programming**
            * template
                * template metaprogramming(TMP)
    * STL
        * template library
            * containers
            * iterators
            * algorithms
            * function objects
